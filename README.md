# Take Home Test for [Artificial](https://artificial.io/)

## Technologies required/used
- [mongodb](https://www.mongodb.com/)
- [npm](https://www.npmjs.com/)
- [nodejs](https://nodejs.org/en/)

## Instructions
Install dependencies:
```
npm install
```

Run with mongodb:
```
npm run mongo
```

Run with json:
```
npm start
# or
npm run json
```
